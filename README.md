## Project Overview  
This project aims to provide both patients and doctors a seamless interface to manage appointment scheduling. The AI Stimulator component analyses doctor availability, patient preferences and historical data to suggest optimal appointment slots — reducing waiting times and improving utilisation of doctor time.

## Features  
- Patient registration and authentication  
- Doctor registration / profile management  
- Search for doctors by speciality, availability and location  
- Booking, rescheduling and cancellation of appointments  
- AI‑driven suggestions:  
  - Suggesting optimal time slots based on doctor’s schedule & past patterns  
  - Avoiding double‑booking/conflicts  
- Calendar view for doctors to view upcoming appointments  
- Notifications (email / in‑app) for confirmations, reminders and cancellations  
- Admin dashboard for oversight (optional)  

## Tech Stack  
- **Frontend:** React.js
- **Backend:** Node.js + Express
